## About AA Tube
AA Tube is an app for watching YouTube on your car's entertainment screen, allowing you to easily access and enjoy your favorite content while driving.
## Installation
* If your phone is working with KingInstaller, you can simply install AA Tube from there.<br/>
0. AA Tube is only compatible with LSPosed Framework, so make sure you have it installed and set up on your Android device.<br/>
1. Root your Android device and install LSPosed Framework.<br/>
2. Download the latest version of AA Tube from the releases page.<br/>
3. Install the APK on your device.<br/>
4. Open LSPosed Manager, find AA Tube in the list of modules, and enable it.<br/>
5. Reboot your device to apply the changes.<br/>
